Estrutura básica do HTML

A estrutura básica do HTML é composta por um conjunto de elementos que formam a "espinha dorsal" de qualquer página web. Ela geralmente começa com a declaração , seguida pela abertura da tag , que contém dois elementos principais: e . O abriga informações como título da página, links para arquivos CSS ou scripts, e metadados. Já o contém o conteúdo visível da página, como textos, imagens, links e outros elementos interativos.

O que são tags?

Tags são os principais componentes do HTML, responsáveis por marcar e estruturar os elementos de uma página web. Elas são escritas entre colchetes angulares, como 'p' para parágrafos e 'a' para links. As tags geralmente possuem uma abertura e um fechamento , mas algumas são auto-fechadas, como 'img'. Cada tag define o tipo de conteúdo ou a funcionalidade que ela representa na página.

Textos

No HTML, o conteúdo textual pode ser estilizado e estruturado com tags específicas. Títulos e subtítulos são representados pelas tags h1 a h6, enquanto parágrafos são definidos com a tag 'p'. Além disso, é possível destacar palavras ou frases usando 'strong' para ênfase forte ou 'em' para ênfase leve. Outros elementos, como 'br' para quebras de linha e 'span' para estilização de trechos, permitem maior controle sobre a formatação do texto.

Listas

As listas no HTML são usadas para organizar itens de forma ordenada ou não ordenada. Listas ordenadas são criadas com a tag 'ol', enquanto listas não ordenadas utilizam 'ul', ambas contendo os itens listados dentro de 'li'. Links, por sua vez, são definidos pela tag 'a', que utiliza o atributo href para especificar o destino. Links podem apontar para outras páginas, seções da mesma página ou até mesmo abrir arquivos e aplicativos externos.
